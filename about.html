About

The purpose of Street Smart is to help users find safe streets to travel on. Using data from data.sfgov.org, Street Smart shows users the streets with the lowest rates of vehicular accidents and speed limit infractions.

Data Analysis

We performed multiple steps to collect, clean, process, and interpret traffic data.

  1. Downloaded accident and speed data from data.sfgov.org
    • The accident data is a log of all accidents happening at intersections from the past 2 years
    • The speed data contains the speed limit and average speed along street segments. Average speeds noticeably higher or lower than speed limit indicate reckless driving or congestion, respectively.
  2. Imported into Google Sheets to delete unnecessary features
  3. Cleaned Speed data:
    • Calculated percent difference between average speed and speed limit using Sheets
    • Calculated average percent difference for all street segments using R
  4. Cleaned Accidents data:
    • Found total people involved in accidents for each intersection
    • Used Sheets pivot tables to find this data for each street
  5. Compiled both features for each street
  6. Normalized values:
    • Normalized accident data to 0-100
    • Normalized speed data to absolute value 0-1
  7. Calculated score:
    • Subtracts normalized accidents from 100
    • If speed data present, scales normalized value by 0.05 and subtracts from 1
    • Then multiplies main result by this scalar
  8. Converted score to tier rating:
    • For even tier distribution given uneven data, only the highest scores are converted to A Tier
    • The score range conversion grows for lower tiers.
  9. With the tier calculated, the data can be displayed to the user on the website!
